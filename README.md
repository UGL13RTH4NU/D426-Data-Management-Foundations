# D426 — Data Management Foundations

Study materials for **WGU D426 — Data Management Foundations**, based on *Fundamentals of Database Management Systems, 3rd Edition*.

These are free, self-contained HTML quizzes and visual guides that run in any browser — no installation needed. Anki decks are also available for download.

## 📋 What's Here

| File | Type | Topic |
|------|------|-------|
| [Crow's Foot Notation Guide](https://ugl13rth4nu.github.io/D426-Data-Management-Foundations/crows_foot_notation_guide.html) | Guide + Quiz | Crow's foot symbols, cardinality & modality, reading ER diagrams (15 questions) |

*More quizzes and Anki decks will be added as the course progresses.*

## 🚀 How to Use

### Online (GitHub Pages)
Click any link above to open the quiz/guide directly in your browser.

### Offline
1. Click the green **Code** button → **Download ZIP**
2. Unzip and open any `.html` file in your browser

### Anki Decks
Download any `.apkg` file from the repo and import it into [Anki](https://apps.ankiweb.net/) (File → Import).

## 📚 Course Topics Covered

- [ ] Database fundamentals & terminology
- [ ] Relational model (tables, rows, columns, keys)
- [ ] SQL sublanguages (DDL, DQL, DML, DCL, DTL)
- [ ] Keys & constraints (PK, FK, referential integrity)
- [ ] DML & query operations (SELECT, INSERT, UPDATE, DELETE)
- [ ] Joins & views
- [ ] Entity-Relationship modeling & crow's foot notation
- [ ] Logical design & normalization (1NF → BCNF)
- [ ] Physical design & indexes
- [ ] Database administration (accounts, security, architecture)
